# Global Frost Overlay Design

**Date:** 2026-03-07

## Goal

Extend the hero-style frosted atmosphere across the whole page background so the site no longer feels like only certain sections have glass haze.

## Constraints

- Keep the change at the top app background layer.
- Do not alter the structure or styling of individual sections.
- Make the overlay much lighter than the existing glass cards so content remains crisp.

## Approved Design

- Add a full-screen fixed frosted overlay between the existing global glow layer and the page content.
- Use a very light translucent gradient plus backdrop blur so the background reads as globally softened rather than fully glass-covered.
- Keep the overlay non-interactive and below all content so buttons, text, and cards stay unchanged.
